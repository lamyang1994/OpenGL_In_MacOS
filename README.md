# OpenGL_In_MacOS

LearnOpenGL : https://learnopengl.com/

TheCherno's Tutorial videos : https://www.youtube.com/watch?v=W3gAzLwfIP0&list=PLlrATfBNZ98foTJPJ_Ev03o2oq3-GGOS2

## Usage

- Compile files, requires **cmake**

```shell
mkdir build && cd build
cmake
make 
```

- Execute

```Shell
cd ../Debug
./bin/App.o
```
